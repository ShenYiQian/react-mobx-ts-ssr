# shinezone-generator-reactts
React V16 + ReactRouter V4 + Mobx + TypeScript + SSR + HMR

# 特性

- [x] React V16 + ReactRouter V4 + Mobx + TypeScript + SSR + HMR + Webpack
- [x] sass模块化
- [x] 支持热更新
- [x] 支持后端渲染
- [x] 支持tslint语法检查,提交检查

# 快速开始

## 安装

````bash
$ npm install
````

## 客户端启动

````bash
$ npm run start
````

## 运行生产环境(开启服务端渲染)

````bash
$ npm run server
````
